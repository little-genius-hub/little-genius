# Little Genius PWA

Little Genius is an educational Progressive Web App (PWA) for children featuring games and stories with bilingual support.

## Features

- Educational games for learning letters and numbers
- Interactive stories
- Bilingual support
- Progressive Web App capabilities for offline use
- Mobile-friendly design

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (v9 or higher)

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/little-genius-pwa.git
cd little-genius-pwa
```

2. Install dependencies

```bash
npm install
```

3. Create a `.env` file based on `.env.example`

```bash
cp .env.example .env
```

4. Start the development server

```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) to view the application in your browser

### Building for Production

```bash
npm run build
npm start
```
